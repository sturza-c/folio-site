INSTALLATION SUR MAC

Folio est téléchargé.
Installez-le en 2 minutes.

Pourquoi macOS affiche un avertissement ?Folio est gratuit, mais n’est pas encore signé ni notarié par Apple. Téléchargez-le uniquement depuis le site officiel Folio.

ÉTAPE 1

Ouvrez l’image disque.

Dans Téléchargements, double-cliquez sur Folio.dmg. Une fenêtre s’ouvre avec Folio et un raccourci vers le dossier Applications.

Si rien ne s’est téléchargé, utilisez « Relancer le téléchargement » en haut.
Folio.dmgImage disque · téléchargement terminé

ÉTAPE 2

Placez Folio dans Applications.

Glissez Folio.app dans le dossier Applications. Folio restera ainsi au bon endroit et apparaîtra dans Spotlight.

Folio→Applications

ÉTAPE 3

Essayez d’ouvrir Folio une fois.

Dans Applications, double-cliquez sur Folio. macOS affiche un avertissement car l’app n’est pas notariée. Fermez cette alerte : cette première tentative fait apparaître le bouton nécessaire dans les Réglages.

Apple ne peut pas vérifier que « Folio » ne contient pas de logiciel malveillant.

Cette action doit être confirmée dans Confidentialité et sécurité.

ÉTAPE 4

Autorisez Folio — une seule fois.

  1. Ouvrez Réglages système.
  2. Choisissez Confidentialité et sécurité.
  3. Descendez jusqu’à Sécurité, puis cliquez sur Ouvrir quand même.
  4. Confirmez avec votre mot de passe ou Touch ID, puis cliquez sur Ouvrir.
Le bouton reste disponible environ une heure après la première tentative d’ouverture.
